# Console For Developers - Our Mall

La consola de desarrolladores se creara con el fin de permitirle a los programadores crear una aplicacion y permitirle acceder a los recursos protegidos de nuestro servidor por medio de OAuth2.0, esta ofrecera la documentacion adecuada para poder realizar dicho proceso y poseera el formulario de registro del cliente, ademas de la pantalla de configuracion como es el consentimiento que se le mostrara al usuario una vez se autorice.
